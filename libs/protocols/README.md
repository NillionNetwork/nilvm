# Cryptographic Protocols
## Cryptographic protocols modeled as state machines

A crate containing all the cryptographic protocols. More specifically:
* [bit_operations](src/bit_operations): Contains bitwise protocols.
* [conditionals](src/conditional): Contains comparisons and conditionals.
* [division](src/division): Contains division, truncation, and modulo protocols.
* [multiplication](src/multiplication): Contains multiplication protocols.
* [random](src/random): Contains protocols for generating random numbers and bits.
* [reveal](src/reveal): Contains the reveal protocol.
